# Выпуклая оболочка (Convex Hull) — интерактивная визуализация

Интерактивное приложение на Python с использованием `matplotlib`, которое позволяет добавлять точки на график и визуализировать построение их выпуклой оболочки с помощью алгоритма Эндрю (монотонной цепочки). Анимация показывает каждый шаг алгоритма: выделение крайних левой и правой точек, разделяющей линии, а также пошаговое построение нижней и верхней оболочек.

##  Демонстрация

![Демонстрация работы алгоритма](gif/proc.gif)

##  Возможности

- Интерактивное добавление точек по клику левой кнопкой мыши
- Пошаговая анимация алгоритма Эндрю
- Визуальное выделение:
  - Крайних левой и правой точек (фиолетовый и малиновый)
  - Разделяющей линии (жёлтый)
  - Нижней оболочки (зелёный)
  - Верхней оболочки (оранжевый)
  - Финальной выпуклой оболочки (красный с заливкой)
- Паузы между шагами для лучшего понимания процесса
- Автоматическая легенда

##  Использование

Запустите скрипт:

```bash
python Andrew_Anisimow.py
```

**Инструкция по управлению:**

1. **Добавление точек:** левый клик мыши на графике
2. **Запуск алгоритма:** нажмите клавишу **Пробел** (должно быть добавлено минимум 3 точки)
3. Наблюдайте за пошаговой анимацией построения выпуклой оболочки

##  Алгоритм

Используется **алгоритм Эндрю** (монотонная цепочка), который работает за O(n log n):

1. Сортировка точек по координате X
2. Выделение крайней левой и правой точек
3. Построение нижней оболочки слева направо
4. Построение верхней оболочки справа налево
5. Объединение оболочек в финальный выпуклый многоугольник

Алгоритм использует кросс-произведение для определения ориентации трёх точек и удаления "невыпуклых" вершин.

##  Примечания

- График имеет фиксированные границы X и Y от 0 до 100
- Для корректной анимации рекомендуется добавлять не менее 5-7 точек
- Все линии и точки автоматически обновляются в процессе построения

##  Возможные проблемы

Если окно с графиком не отвечает или не отображается корректно, убедитесь, что:
- Установлена поддерживаемая версия matplotlib
- Используется интерактивный бэкенд (например, TkAgg или Qt5Agg)

Для проверки бэкенда выполните в Python:

```python
import matplotlib
print(matplotlib.get_backend())
```
